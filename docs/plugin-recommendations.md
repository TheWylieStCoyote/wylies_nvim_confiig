## Priority Recommendations

### Essential (High Impact)

1. **neotest** - Unified test running across all languages
2. **harpoon** - Must-have for fast file navigation
3. **diffview.nvim** - Superior git diff experience
4. **vim-dadbod-ui** - Full database client
5. **oil.nvim** - Intuitive file management

### Nice to Have (Medium Impact)

1. **neogen** - Auto-generate docstrings
2. **refactoring.nvim** - Extract functions, variables
3. **rest.nvim** - API testing without leaving editor
4. **overseer.nvim** - Task running and build systems
5. **flash.nvim** - Enhanced motions

### Specialized (Use Case Dependent)

1. **obsidian.nvim** - If you use Obsidian for notes
2. **remote-nvim.nvim** - If you work on remote servers
3. **codecompanion.nvim** - If you want multiple AI providers
4. **neorg** - If you want org-mode style organization
5. **octo.nvim** - If you manage GitHub issues/PRs frequently

## Installation Notes

Most plugins can be added to `lua/plugins/` as a new file:

```lua
-- lua/plugins/example.lua
return {
  {
    "author/plugin-name",
    dependencies = { "dependency/plugin" },
    opts = {
      -- configuration
    },
  },
}
```

For LazyVim, check if there's an [extra](http://www.lazyvim.org/extras) available first, as these are pre-configured.
