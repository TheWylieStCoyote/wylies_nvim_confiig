# Future Additions

Potential plugins and features to consider adding.

## Navigation & Workflow

| Plugin | Description |
|--------|-------------|
| harpoon | Quick bookmarks for frequently used files |
| neogit | Full git interface (like magit for Emacs) |
| diffview.nvim | Better git diff/merge visualization |

## Editing

| Plugin | Description |
|--------|-------------|
| nvim-autopairs | Auto-close brackets, quotes, parens |
| nvim-surround | Add/change/delete surrounding chars (`cs"'` changes `"text"` to `'text'`) |
| mini.ai | Enhanced text objects (function arguments, quotes, etc.) |

## Visual

| Plugin | Description |
|--------|-------------|
| indent-blankline | Show indentation guides |
| todo-comments | Highlight TODO/FIXME/NOTE comments |

## Quality of Life

| Plugin | Description |
|--------|-------------|
| undotree | Visual undo history tree |
| better-escape | Use `jk` to exit insert mode instead of `<Esc>` |
